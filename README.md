# Task AI

A powerful, aesthetic Todo application designed for maximum productivity with a unique dual-task system.

## ✨ Key Features

- **Dynamic Tasks (Daily Core):** Focus on your essential "Primary" tasks that repeat daily or on custom schedules.
- **Random Tasks (Backlog):** Organize your "High", "Middle", and "Low" priority tasks in a persistent backlog.
- **Circular Queue System:** Secondary and Tertiary tasks rotate to the back of the queue when completed or skipped, ensuring a fresh focus every day without losing track of your "evergreen" list.
- **Queue Backlog UI:** A dedicated, collapsible section to see the full "turn order" of your queued tasks.
- **Premium Aesthetics:** Dark mode support, glassmorphism UI, smooth animations, and celebratory effects (Confetti & Sounds) on task completion.
- **Daily Reset:** Automatic daily reset for primary tasks to keep your focus sharp.

## 🚀 Priority System

- **Primary:** Core daily tasks (Recurrency: Daily, Every X Days, Weekly).
- **Secondary/Tertiary:** Queued tasks that rotate through the "Up Next" slots.
- **Random (High/Middle/Low):** Standalone backlog tasks sorted by priority and age.

## 🛠️ Technology Stack

- **Frontend:** React + Vite + TypeScript
- **Styling:** Tailwind CSS + Framer Motion
- **State Management:** Zustand (with Persistence)
- **Utilities:** Date-fns, Lucide Icons, React Hot Toast
